# Python运算符

1、算术运算符：+ 、- 、*、 /、 //、 % 、**

2、赋值运算符：=、 +=、 -=、 *=、 /=、 %=、 //=、 **=

3、比较运算符：== 、！=、 <、>、<=、 >=

4、逻辑运算符：and、or、not
